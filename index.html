Projetos Selecionados

Otimizador de Custo de Energia / Java e IA (2023–2024)

Objetivo: Desenvolver e implementar uma ferramenta inteligente para otimizar custos residenciais de energia usando heurísticas e simulações tarifárias.

Meu papel: Implementei Algoritmo Genético, GRASP, GRASP Reativo e Metaheurística Híbrida; escrevi documentação técnica e registrei o software no INPI.

Tecnologias: Java, Python, heurísticas de IA, gerenciamento pelo lado da demanda (Demand-side management), UML.

Resultado / impacto: Software registrado no INPI (BR512024001218-4); comprovou redução real em cenários simulados; base técnica e experimental da dissertação.

Knowledge Graph do Modelo Conceitual do Domínio (2022–2023)

Objetivo: Construir e formalizar um grafo de conhecimento interoperável para organizar dados de TCCs stricto sensu segundo princípios da Web Semântica e padrões OWL/RDF.

Meu papel: Defini classes e relações; implementei e validei dados ligados de dissertação.

Tecnologias: OWL, RDF, SPARQL, GraphDB, CmapTools.

Resultado / impacto: Produziu grafo funcional e alinhado a padrões web; permitiu consultas complexas e aumentou precisão semântica; base do trabalho acadêmico.

Ontologia de Referência e Modelo de Design: Domínio Defesa (2022)

Objetivo: Modelar uma ontologia de referência e um modelo de design para estruturar elementos conceituais de TCCs com semântica formal padronizada.

Meu papel: Concepção, modelagem, validação das classes e relacionamentos, documentação técnica.

Tecnologias: OWL, RDF, SPARQL, GraphDB, Protegé, VocBench.

Resultado / impacto: Entregou artefato de padronização semântica, integrando a dissertação e permitindo consultas mais estruturadas.
